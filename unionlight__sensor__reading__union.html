Sample code illustrating communications with the TSC-34725 via I2C, using a bus-pirate compatible I2C library. Simple data structure which defines a two-byte piece of data.

This "union" allows us to access the two-byte data into two ways: as a pair of low and high bytes (var.low and var.high), or as a single 16-bit unsigned integer (var.full).

This is convenient for intepreting our light sensor data!
